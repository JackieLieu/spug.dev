`Spug` 是面向中小型企业设计的无 Agent 的自动化运维平台，整合了命令远程批量执行、应用发布、任务计划、配置中心、监控、报警等一系列功能。

## 特性

- 命令远程批量执行
- 浏览器在线命令终端
- 灵活可自定义的应用发布
- 直观易维护的任务调度系统
- 灵活强大的配置中心
- 简单易用的监控报警
- 基于 [Ant Design](https://ant-design.gitee.io/) 的漂亮UI
- 前后端完全开源
